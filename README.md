# 📦 Proyecto Next.js - Guía de instalación y uso

## 🚀 Requisitos previos

Antes de comenzar, asegúrate de tener instalado **pnpm**.

Si no lo tienes instalado, puedes hacerlo con:

```bash
npm install -g pnpm
```

---

## 📥 Instalación de dependencias

Una vez tengas pnpm disponible, instala las dependencias del proyecto con:

```bash
pnpm install
```

---

## ▶️ Ejecutar el proyecto

Para iniciar el entorno de desarrollo:

```bash
pnpm dev
```

O también puedes usar:

```bash
pnpm start
```

---

## 🏗️ Construir el proyecto

Para generar la build de producción:

```bash
pnpm build
```

---

## 🧪 Ejecutar pruebas unitarias

El proyecto cuenta con pruebas unitarias configuradas con Jest.

Para ejecutarlas:

```bash
pnpm test
```

---

## 📌 Notas adicionales

* Asegúrate de tener una versión reciente de Node.js instalada.
* Se recomienda usar **pnpm** para mantener consistencia en la gestión de dependencias.
* Las pruebas están separadas por frontend y backend para mejor organización.
